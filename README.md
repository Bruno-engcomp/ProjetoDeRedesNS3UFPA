## Projeto da disciplina de redes de computadores do curso de Engenharia da computação da UFPA

# Objetivo
Ao final deste projeto, o aluno deve ter implementado (através do simulador ex. ns-
2, NS-3, OPNET, OMNET, etc) uma topologia de rede de computadores contendo
tráfego(s) (ex. FTP, CBR) entres os nos da rede (mínimo 10 nos), devendo
demonstrar conhecimentos sobre:
Introdução a Simulação;
Ferramentas de Simulação;
Topologias de Rede de Computadores (lógica e física);
Modelo de Tráfego;
